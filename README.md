# Image Labeler — VS Code 图片标注扩展

高效图片数据标注工具，专为视频抽帧数据集设计。支持**保留/删除**二分类标注，三种标注模式覆盖不同场景。

## 功能特性

- 📂 **侧边栏浏览** — TreeView 展示文件夹树和图片列表，实时显示标注进度和状态图标
- 🖼️ **大图预览标注** — 单张全屏查看，键盘快捷键快速标注，支持 30〜120fps 高速操作
- 🎬 **播放模式** — 按指定 FPS 自动播放帧序列，默认保留/删除，遇到需要改变的帧按键反选
- 🗂️ **缩略图网格** — 网格视图批量浏览，支持 Shift/Ctrl 多选批量标注、按状态筛选
- 💾 **自动持久化** — 每子文件夹独立 `.label-state.json` 存储，延迟批量写盘不阻塞标注
- 📤 **结果导出** — 标注结果导出为 CSV 文件
- 🔄 **跨文件夹自动跳转** — 当前文件夹标注完成后自动进入下一文件夹

## 安装

### 方式一：从 VSIX 安装

1. 确保已安装 [Node.js](https://nodejs.org/)
2. 克隆项目并编译打包：

```bash
git clone https://github.com/ppknkn/vscode-image-labeler.git
cd vscode-image-labeler
npm install
npm run compile
npm install -g @vscode/vsce
vsce package
```

3. 在 VS Code 中按 `Ctrl+Shift+P`，输入 "Extensions: Install from VSIX..."，选择生成的 `.vsix` 文件

### 方式二：开发调试

```bash
npm install
npm run compile
```

然后在 VS Code 中按 `F5` 启动扩展开发宿主窗口。

## 使用方法

### 1. 打开图片文件夹

在 VS Code 中打开包含图片的文件夹，点击活动栏的图片图标展开"图片标注"面板。

支持的图片格式：`jpg` `jpeg` `png` `gif` `bmp` `webp`

### 2. 标注图片

点击侧边栏中的图片，在图片查看器中通过快捷键标注：

| 快捷键 | 操作 |
|--------|------|
| `K` 或 `→` | 标记为**保留**，跳到下一张未标注 |
| `D` 或 `←` | 标记为**删除**，跳到下一张未标注 |
| `Space` | 切换保留/删除，跳到下一张 |
| `↑` | 上一张（不改变标注） |
| `↓` | 下一张（不改变标注） |
| `P` | 播放/暂停 |

也可点击底部按钮或右键菜单操作。

### 3. 播放模式（视频帧标注利器）

对于视频抽帧数据集，推荐使用播放模式：

1. 在底部播放控制栏选择 **FPS**（1〜120）
2. 选择**默认行为**：保留（绿色）或删除（红色）
3. 点击 **▶ 播放** 或按 `P` 键
4. 图片按指定帧率自动播放，每帧自动标记为默认值
5. 看到需要改变的帧时，按下**反选键**：

| 默认行为 | 反选键 | 效果 |
|----------|--------|------|
| 保留 | `D` | 把当前帧标记为删除 |
| 删除 | `K` | 把当前帧标记为保留 |

> 💡 播放期间按 `Space` 暂停，按 `P` 恢复，或直接点击暂停按钮。

### 4. 缩略图网格批量操作

- 在侧边栏右键点击文件夹 → "打开缩略图网格"
- 单击选择，`Ctrl+Click` 追加，`Shift+Click` 范围选择
- 选中后使用底部浮动工具栏批量标注
- 使用顶部筛选栏按状态筛选

### 5. 导出结果

点击侧边栏工具栏的导出按钮（💾），生成 CSV 文件。

## 标注状态文件

每个包含图片的子文件夹下生成 `.label-state.json`：

```json
{
  "version": 1,
  "folder": "/path/to/images",
  "lastUpdated": "2026-07-04T12:00:00.000Z",
  "files": {
    "frame_0001.jpg": "keep",
    "frame_0002.jpg": "delete",
    "frame_0003.jpg": null
  }
}
```

- `"keep"` = 保留
- `"delete"` = 删除
- `null` = 未标注
